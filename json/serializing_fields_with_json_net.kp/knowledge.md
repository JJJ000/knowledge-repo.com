**Contents**

[TOC]

1. **Objects**
   - Objects are represented as name/value pairs. The names must be strings and the values can be any valid JSON value.

2. **Arrays**
   - Arrays are represented as an ordered list of values. The values can be any valid JSON value.

3. **Strings**
   - Strings are represented as a sequence of zero or more Unicode characters.

4. **Numbers**
   - Numbers are represented as double-precision floating-point values.

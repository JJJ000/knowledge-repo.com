# json.dump()
json.dump() is used to write JSON objects to a file. It takes two arguments:
1. The JSON object to be written
2. The file object to which the JSON object is written

# json.dumps()
json.dumps() is used to convert a Python object into a JSON string. It takes one argument:
1. The Python object to be converted into a JSON string

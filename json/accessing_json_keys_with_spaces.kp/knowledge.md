#### Solution 1:

We can access the keys having spaces in JSON object by enclosing them in quotes.

#### Solution 2:

We can also use the bracket notation to access keys having spaces in JSON object. For example, if the key is `"my key"`, then we can access it as `jsonObj["my key"]`.

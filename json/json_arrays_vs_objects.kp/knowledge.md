### Arrays

JSON Arrays are used to store multiple values in a single object. These values can be of any data type, including strings, numbers, objects, and even other arrays. Arrays are indicated by square brackets and the values are separated by commas.

### Objects

JSON Objects are used to store key-value pairs. The key is used to identify the value and the value can be of any data type. Objects are indicated by curly brackets and the key-value pairs are separated by commas.

### Comparison

JSON Arrays are used to store multiple values in a single object, while JSON Objects are used to store key-value pairs. Arrays are indicated by square brackets and values are separated by commas, while Objects are indicated by curly brackets and the key-value pairs are separated by commas. Arrays cannot have duplicate values, while Objects can have duplicate keys.

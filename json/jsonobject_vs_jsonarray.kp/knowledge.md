**Contents**

[TOC]

1. JSONObject:
- JSONObject is a class that is used to parse the JSON data. It is used to represent the data in a key-value pair format. It is an unordered collection of key-value pairs.

2. JSONArray:
- JSONArray is a class that is used to parse the JSON data. It is a collection of values that are enclosed in square brackets. It represents an ordered list of values.

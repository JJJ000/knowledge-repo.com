**Valid Characters**

- Letters: A-Z, a-z
- Numbers: 0-9
- Special Characters: _ (underscore), $ (dollar sign)

**Invalid Characters**

- Spaces
- Special Characters: !, @, #, %, ^, &, *, (, ), +, -, =, [, ], {, }, :, ;, ", ', <, >, ?, /

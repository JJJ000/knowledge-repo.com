# Section 1: Copy JSON

1. Select the JSON you want to format.
2. Copy the JSON by pressing `Ctrl+C` (Windows) or `Command+C` (Mac).

# Section 2: Paste JSON

1. Open the file you want to paste the JSON into.
2. Place the cursor in the location you want to paste the JSON.
3. Paste the JSON by pressing `Ctrl+V` (Windows) or `Command+V` (Mac).

# Section 3: Format JSON

1. Select the JSON you just pasted.
2. Press `Ctrl+Alt+L` (Windows) or `Ctrl+Option+L` (Mac) to format the JSON.

# Section 4: Check Formatting

1. Check the JSON to make sure it is formatted correctly.
2. If the formatting is incorrect, repeat steps in Section 3.

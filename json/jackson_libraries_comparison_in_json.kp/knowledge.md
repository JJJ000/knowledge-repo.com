# org.codehaus.jackson

org.codehaus.jackson is the original Jackson library, written by the founder of the Jackson project, Tatu Saloranta. It is the basis of all other Jackson libraries, including com.fasterxml.jackson.core. The code is open-source and is hosted on GitHub.

# Features

org.codehaus.jackson offers a wide range of features, including:
- Support for reading and writing JSON
- Support for Java and Scala
- Support for streaming API
- Support for data binding
- Support for tree model
- Support for annotations
- Support for data format modules
- Support for custom serializers and deserializers

# Pros

- Easy to use
- Open source
- Well-maintained
- Good documentation

# Cons

- No support for other languages
- Limited support for data formats
- No support for custom serializers and deserializers

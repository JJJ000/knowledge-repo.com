## Step 1: 
Open the VS Code settings by pressing `Ctrl + ,` or `F1` and type `settings`. 

## Step 2: 
In the search bar, type `json` and select `JSON: Comments`.

## Step 3: 
Toggle the `JSON: Comments` option to `off`.

## Step 4: 
Save the settings and restart VS Code. The error message should now be disabled.

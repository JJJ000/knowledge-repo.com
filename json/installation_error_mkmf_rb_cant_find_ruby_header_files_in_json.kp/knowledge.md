# Solution 1

1. Check if the correct version of ruby is installed

2. Check if the correct version of the ruby-dev package is installed

3. Check if the correct version of the json gem is installed

# Solution 2

1. Check if the correct version of ruby is installed

2. Try to reinstall the ruby-dev package

3. Reinstall the json gem

4. Check if the headers files for ruby are in the correct directory

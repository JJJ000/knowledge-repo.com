## Definition 
Object: An object is an unordered set of name/value pairs.
Hash: A hash is an unordered set of key/value pairs.

## Structure
Object: An object is structured using curly braces and contains one or more name/value pairs.
Hash: A hash is structured using curly braces and contains one or more key/value pairs.

## Syntax 
Object: An object is written in the following syntax:
```
{
  name1: value1,
  name2: value2,
  ...
}
```
Hash: A hash is written in the following syntax:
```
{
  key1: value1,
  key2: value2,
  ...
}
```

## Usage
Object: An object is used to store data in a structured way.
Hash: A hash is used to store data in an unordered way.

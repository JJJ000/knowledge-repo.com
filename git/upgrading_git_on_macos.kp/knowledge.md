### Section 1 - Prerequisites
Before upgrading Git, it is important to make sure that your system meets the following prerequisites:

- A Mac running macOS 10.10 or higher
- Command Line Tools for Xcode installed

### Section 2 - Checking Current Version
To check the current version of Git installed on your system, open the terminal and run the following command:

```
git --version
```

### Section 3 - Installing Git
To install the latest version of Git, open the terminal and run the following command:

```
brew install git
```

### Section 4 - Verifying Installation
To verify the installation of the latest version of Git, open the terminal and run the following command:

```
git --version
```

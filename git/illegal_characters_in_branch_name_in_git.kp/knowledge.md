**Contents**

[TOC]

1. Reserved Characters:
- `/`
- `\`
- `.`
- `:`

2. Control Characters:
- `~`
- `^`
- `*`
- `?`
- `[`
- `]`

3. Whitespace Characters:
- ` ` (space)
- `\t` (tab)
- `\n` (newline)
- `\r` (carriage return)

4. Other Characters:
- `@`
- `#`
- `$`
- `!`

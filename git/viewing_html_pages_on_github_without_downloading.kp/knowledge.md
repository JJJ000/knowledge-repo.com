### Viewing HTML Pages on Github

### Using Rawgit
1. Visit [Rawgit](https://rawgit.com/)
2. Enter the URL of the HTML file from Github
3. Click "Generate"
4. Copy the URL of the generated page
5. Paste the URL into your browser

### Using Github Pages
1. Create a new branch in your repository called `gh-pages`
2. Add your HTML file to the `gh-pages` branch
3. Visit `https://username.github.io/repository-name/file-name.html` in your browser

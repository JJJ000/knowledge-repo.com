### Using Git Log

1. Open the terminal and navigate to the root directory of the repository.
2. Type `git log` to view the commit history.
3. Select the commit for which you want to get the hash.
4. Copy the hash value of the commit.

### Using Git Show

1. Open the terminal and navigate to the root directory of the repository.
2. Type `git show` to view the last commit.
3. Copy the hash value of the commit.

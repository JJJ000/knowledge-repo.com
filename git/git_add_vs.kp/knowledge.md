#### Git Add *

Git Add * adds all files in the current working directory and its subdirectories to the staging area. It is a shorthand for "git add ." and is used to add all changes to the current working directory and its subdirectories to the staging area.

#### Git Add .

Git Add . adds all files in the current working directory to the staging area. It is used to add all changes to the current working directory to the staging area. This is the same as running the command "git add *" but it is more explicit and can be used to add specific files to the staging area.

**Contents**

[TOC]

1. Exit VIM
  - To exit VIM, press `Esc` to enter command mode, then type `:q` and press `Enter` to exit.

2. Return to Terminal
  - After exiting VIM, press `Ctrl + c` to return to the terminal.

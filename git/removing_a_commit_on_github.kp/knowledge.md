### Deleting a Commit
1. On the main page of the repository, click on the **Commits** tab.
2. Find the commit you want to delete and click the **...** button next to it.
3. Select **Revert "Commit Name"** from the dropdown menu.
4. On the next page, click **Confirm**.

### Reverting a Commit
1. On the main page of the repository, click on the **Commits** tab.
2. Find the commit you want to revert and click the **...** button next to it.
3. Select **Revert "Commit Name"** from the dropdown menu.
4. On the next page, click **Commit Changes**.
5. Push the changes to the remote repository.

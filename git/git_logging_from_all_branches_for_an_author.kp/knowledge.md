**Contents**

[TOC]

### Set Up
1. Open a terminal window.
2. Navigate to the local repository.

### Run the Log Command
1. Enter the following command in the terminal window:

```git
git log --all --author="<author>"
```

Replace `<author>` with the author's name.

### View the Log
1. The terminal window will display the log entries for the specified author.

### Close the Log
1. To close the log, press `CTRL+C` in the terminal window.

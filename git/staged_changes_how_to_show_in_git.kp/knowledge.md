#1 Viewing Staged Changes

The `git diff --staged` command can be used to view the changes which have been staged in Git. This command will show the differences between the staged version of a file and the version that was last committed.

#2 Using the `git status` Command

The `git status` command can also be used to view the changes which have been staged in Git. This command will show a list of all the files which have been modified and/or staged for commit.

#3 Viewing the Commit Log

The `git log` command can be used to view the commit log, which will show all the commits that have been made in the repository. This can be used to view the changes which have been staged and committed.

#4 Using the `git show` Command

The `git show` command can be used to view the changes which have been staged and committed. This command will show the differences between the committed version of a file and the version that was last committed.

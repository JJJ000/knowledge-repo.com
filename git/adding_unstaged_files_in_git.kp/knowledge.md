# Add All Untracked Files

1. Navigate to the root directory of the repository.
2. Run the command `git add .`
3. This will add all untracked files to the staging area.

# Add Specific Untracked Files

1. Navigate to the root directory of the repository.
2. Run the command `git add <file_name>`
3. This will add the specific untracked file to the staging area.

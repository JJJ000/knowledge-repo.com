# Using the Command Line
1. Open the Command Prompt.
2. Navigate to the file you want to change the case of.
3. Type `rename <oldname> <newname>` and press Enter.

# Using the File Explorer
1. Open the File Explorer.
2. Navigate to the file you want to change the case of.
3. Right-click on the file and select Rename.
4. Type in the new name with the desired case and press Enter.

### Shortcut

The quickest way to exit the results of `git diff` in Git Bash on Windows is to use the `Ctrl+C` shortcut.

### Command

Alternatively, you can type the `q` command to exit the results of `git diff`.

### Exit Button

You can also click the “X” button in the top right corner of the window to exit the results.

### Close Terminal

Finally, you can close the terminal window by clicking the “X” button in the top right corner of the terminal window.

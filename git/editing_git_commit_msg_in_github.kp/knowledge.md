# How to Edit a Git Commit Message on GitHub

## Step 1: Navigate to the Repository

First, navigate to the repository that contains the commit you would like to edit.

## Step 2: Select the Commit

Once you have navigated to the repository, select the commit you would like to edit.

## Step 3: Edit the Commit Message

Once you have selected the commit, you can edit the commit message. Make sure to save your changes once you are finished.

## Step 4: Push the Changes

Finally, push the changes to the repository to save your new commit message.

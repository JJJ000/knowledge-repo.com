**Git Log**

- `gl`  : `git log`
- `glo` : `git log --oneline --decorate --all --graph`

**Git Checkout**

- `gc`  : `git checkout`
- `gcm` : `git checkout master`

**Git Commit**

- `gca` : `git commit -a`
- `gcm` : `git commit -m`

**Git Push**

- `gp`  : `git push`
- `gpo` : `git push origin`

### Using Command Line
1. Navigate to the local repository in your terminal.
2. Enter the command `git branch`.
3. The current branch will be listed with an asterisk next to it.

### Using the GitHub Desktop App
1. Open the GitHub Desktop App.
2. Select the local repository.
3. The current branch will be listed in the bottom left corner.

# Using GitHub UI
1. Go to the repository page
2. Select the branch you want to compare from the branch dropdown menu
3. Select the branch you want to compare against from the branch dropdown menu
4. Click the "Compare" button

# Using the Command Line
1. Change the working directory to the local repository
2. Run the command `git diff <branch1> <branch2>`
3. View the diff output in the terminal

**Contents**

[TOC]

1. Using Keyboard Shortcuts 
    - On Windows: `Ctrl + X` 
    - On Mac: `Cmd + X`

2. Using the Command Line 
    - Type `:wq` and press `Enter`

3. Using the GUI 
    - Click the `X` in the top right corner of the window

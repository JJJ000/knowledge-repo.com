### Save

To save a file in Vim for Windows, type the command `:w` and press Enter.

### Exit

To exit a file in Vim for Windows, type the command `:q` and press Enter.

### Save and Exit

To save and exit a file in Vim for Windows, type the command `:wq` and press Enter.

### Exit Without Saving

To exit a file without saving in Vim for Windows, type the command `:q!` and press Enter.

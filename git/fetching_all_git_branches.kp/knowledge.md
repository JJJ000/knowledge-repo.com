### Locally
1. Open the command line and navigate to your local repository.
2. Type `git branch` to list all the branches in your local repository.

### Remotely
1. Open the command line and navigate to your local repository.
2. Type `git branch -a` to list all the branches in your remote repository.

### List Remote Branches
1. Open the command line and navigate to your local repository.
2. Type `git branch -r` to list all the remote branches in your repository.

### List All Branches
1. Open the command line and navigate to your local repository.
2. Type `git branch -a` to list all the branches in your local and remote repositories.

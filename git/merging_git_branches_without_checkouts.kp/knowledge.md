### Merge
To merge a branch without using checkouts, you can use the `git merge` command. This command will take the specified branch and merge it into the current branch.

### Update
To update a branch without using checkouts, you can use the `git fetch` command. This command will fetch any new commits from the specified branch and add them to the local repository.

### Pull
To pull a branch without using checkouts, you can use the `git pull` command. This command will fetch any new commits from the specified branch and then merge them into the current branch.

**Contents**

[TOC]

####Using Git Bash
1. Open Git Bash.
2. Type in the following command: `git stash clear`.
3. Press enter.

####Using Command Line
1. Open Command Line.
2. Type in the following command: `git stash clear`.
3. Press enter.

####Using Visual Studio Code
1. Open Visual Studio Code.
2. Open the Command Palette (Ctrl + Shift + P).
3. Type in `Git: Stash Clear` and press enter.

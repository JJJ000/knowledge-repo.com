**Contents**

[TOC]

### Setting Up

1. Open the repository in your terminal. 
2. Navigate to the sub-directory of the repository.

### Using `git log` 

1. Use the command `git log` to view the commit history specific to the sub-directory.

### Using `git log -- <path>` 

1. Use the command `git log -- <path>` to view the commit history specific to the sub-directory.

### Using `git log -- <path> -- <file>` 

1. Use the command `git log -- <path> -- <file>` to view the commit history specific to a particular file in the sub-directory.

## Step 1: Install Homebrew

1. Open the Terminal app and type `/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`
2. Press `Return` and type your password when prompted
3. Follow the instructions on the screen

## Step 2: Install ImageMagick

1. In the Terminal app, type `brew install imagemagick`
2. Press `Return`
3. Wait for the installation to complete

## Step 3: Verify Installation

1. In the Terminal app, type `magick --version`
2. Press `Return`
3. Verify that the version of ImageMagick is correct

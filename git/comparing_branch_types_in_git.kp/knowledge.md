### Local Branch
A local branch is a branch created and stored in the local repository. It is not visible to other remote users.

### Local Tracking Branch
A local tracking branch is a local branch that has a direct relationship to a remote branch. This allows the local branch to track changes made to the remote branch.

### Remote Branch
A remote branch is a branch created and stored in a remote repository. It is visible to all remote users.

### Remote Tracking Branch
A remote tracking branch is a remote branch that has a direct relationship to a local branch. This allows the remote branch to track changes made to the local branch.

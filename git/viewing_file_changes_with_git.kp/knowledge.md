### git log

To view the changes made to a specific file using git, you can use the `git log` command. This command will show a list of all the commits made to the file, along with the author, date, and a description of the changes.

### git show

The `git show` command can be used to view the changes made to a specific file in more detail. This command will show the differences between the current version of the file and the previous version.

### git diff

The `git diff` command can also be used to view the changes made to a specific file. This command will show the differences between the current version of the file and the previous version, but in a more detailed format.

### git blame

The `git blame` command can be used to view the changes made to a specific file and who made them. This command will show the author of each line of the file, along with the date it was last modified.

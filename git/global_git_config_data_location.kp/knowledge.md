**Contents**

[TOC]

1. **Local Machine**
- The global git configuration data is stored in a file called `.gitconfig` which is located in the user's home directory.

2. **Windows**
- On Windows, the `.gitconfig` file is located in the `C:\Users\[username]` directory.

3. **Mac/Linux**
- On Mac and Linux systems, the `.gitconfig` file is located in the `~` directory.

4. **GitHub**
- The global git configuration data can also be stored in the `Settings` tab of the GitHub account.

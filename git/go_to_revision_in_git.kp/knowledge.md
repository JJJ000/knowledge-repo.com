### Checkout revision

To go to a particular revision in Git, you can use the `git checkout` command with the revision number as an argument. This will move your current HEAD to the specified revision.

### Checkout branch

Alternatively, you can also check out a branch that contains the desired revision. This can be done with the `git checkout` command followed by the branch name.

### Checkout tag

You can also check out a tag that contains the desired revision. This can be done with the `git checkout` command followed by the tag name.

### Checkout commit

Finally, you can also check out a commit that contains the desired revision. This can be done with the `git checkout` command followed by the commit hash.

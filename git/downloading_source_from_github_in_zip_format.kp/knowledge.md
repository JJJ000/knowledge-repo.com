### Section 1 - Accessing the Repository
1. Go to the GitHub repository you want to download.
2. Click on the green "Code" button on the right side of the page.

### Section 2 - Downloading the ZIP File
1. Click the "Download ZIP" button on the bottom right of the page.
2. A zip file of the repository will begin downloading.

### Section 3 - Extracting the ZIP File
1. Locate the zip file on your computer.
2. Right-click on the file and select "Extract All".
3. Choose the location you want to extract the files to.

### Section 4 - Viewing the Files
1. After the extraction is complete, open the folder where you extracted the files.
2. You should now see all the files from the GitHub repository.

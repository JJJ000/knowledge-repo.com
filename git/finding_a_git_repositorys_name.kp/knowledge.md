# Section 1: Using Git Command Line
1. Navigate to the local Git repository in the terminal.
2. Execute the command `git config --local --get remote.origin.url`
3. The repository name will be displayed in the output.

# Section 2: Using Github UI
1. Log in to Github.
2. Navigate to the repository.
3. The repository name will be displayed in the top-left corner of the page.

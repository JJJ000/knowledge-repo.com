## git status
The `git status` command is used to list all of the files that are currently staged in the local repository.

## git diff
The `git diff` command is used to compare the differences between the staged files and the files in the working directory.

## git add
The `git add` command is used to add files to the staging area.

## git commit
The `git commit` command is used to commit the staged files to the local repository.

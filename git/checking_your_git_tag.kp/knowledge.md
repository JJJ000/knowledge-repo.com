### Git Tag

### Check Tag

To check which git tag you are on, you can use the following command:

`git tag`

This will list all of the tags that have been applied to the current branch.

### View Tag

To view the details of a specific tag, you can use the following command:

`git show <tagname>`

This will show the commit message, author, date, and other details associated with the tag.

### Checkout Tag

If you want to checkout a specific tag, you can use the following command:

`git checkout <tagname>`

This will switch your working branch to the tag specified.

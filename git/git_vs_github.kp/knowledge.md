### Git
Git is a distributed version control system. It is a piece of software that tracks changes made to files, allowing developers to collaborate on projects, keep track of their changes, and revert back to any previous version. 

### GitHub
GitHub is a web-based hosting service for version control using Git. It provides access control and several collaboration features such as bug tracking, feature requests, task management, and wikis for every project. It also provides a web interface for users to view, manage and share their projects.

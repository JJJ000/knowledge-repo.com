# Section 1: Locating the Commits
1. Open the repository in your favorite Git client.
2. Locate the two commits you want to compare.

# Section 2: Viewing the Changes
1. Select the two commits you want to compare.
2. Click the “View Changes” button to view the differences between the two commits.

# Section 3: Listing the Files
1. Look for the “Files Changed” section, which will list all of the files that have been changed between the two commits.
2. Click the “List Files” button to view a list of all the files that have been changed between the two commits.

# Section 4: Exporting the List
1. Once you have the list of files, you can export it as a text file or CSV file.
2. This will give you a complete list of all the files that were changed between the two commits.

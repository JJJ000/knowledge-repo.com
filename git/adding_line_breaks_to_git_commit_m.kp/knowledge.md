### Solution

### Step 1

Type the command `git commit -m` followed by the message you want to include in the commit.

### Step 2

Press `Shift + Enter` to add a line break to the message.

### Step 3

Type the next line of the message.

### Step 4

Press `Enter` to finish the commit message.

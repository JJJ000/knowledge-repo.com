# Using Git Log

1. Navigate to the repository you want to search.
2. Run the following command: `git log --author="<username>"`
3. This will list all commits by the specified user.

# Using GitHub

1. Navigate to the repository you want to search.
2. Click on the "Commits" tab.
3. Enter the username in the search box.
4. This will list all commits by the specified user.

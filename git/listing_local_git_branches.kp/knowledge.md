### Using git branch

1. Open the terminal window.
2. Change the directory to the local repository.
3. Execute the command `git branch` to list all the local branches.

### Using git branch -l

1. Open the terminal window.
2. Change the directory to the local repository.
3. Execute the command `git branch -l` to list only the local branches.

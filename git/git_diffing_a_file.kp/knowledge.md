### Using the Command Line

1. Navigate to the directory containing the file.
2. Use the command `git diff <file name>` to view the differences between the current version and the last commit.

### Using a Graphical Tool

1. Open the repository in a graphical tool such as SourceTree.
2. Select the file and view the "Diff" tab to see the differences between the current version and the last commit.

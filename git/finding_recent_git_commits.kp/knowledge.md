### Using the Command Line
1. Navigate to the repository in your terminal.
2. Use the command `git log [filename]` to view a list of commits that modified the specified file.
3. The most recent commit will be at the top of the list.

### Using the GitHub Website
1. Navigate to the repository on the GitHub website.
2. Click on the "Commits" tab.
3. Click on the "Files changed" tab.
4. Select the file you want to view.
5. The most recent commit will be at the top of the list.

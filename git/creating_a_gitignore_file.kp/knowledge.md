**Contents**

[TOC]

1. Create the .gitignore File 
   - Create a new file in the root directory of your project and name it `.gitignore`.
   - Open the file in a text editor.

2. Add Rules to the File 
   - Add the rules for the files and directories that you want to ignore.
   - All rules should be on separate lines.

3. Save the File 
   - Save the file in the root directory of your project.

4. Add the File to the Repository 
   - Add the file to your repository using the `git add` command.
   - Commit the file using the `git commit` command.

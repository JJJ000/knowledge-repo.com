**Contents**

[TOC]

1. Syntax
- `#ISSUE-NUMBER`

2. Example
- `git commit -m "Fixed issue #123"`

3. Explanation
- The `#` symbol is used to reference an issue number on GitHub within a commit message.

4. Note
- The issue number must be preceded by the `#` symbol.

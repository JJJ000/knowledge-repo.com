**Contents**

[TOC]

### Set Up Your Commit
1. Change to the directory of the repository you want to commit to.
2. Use the `git add` command to stage the files you want to commit.

### Create the Commit
1. Use the `git commit` command to create your commit.
2. Enter your commit message in the command line.

### Add a Description
1. Use the `git commit -v` command to open up a text editor window.
2. Enter your commit description in the text editor window.

### Finalize Your Commit
1. Save and close the text editor window.
2. Your commit will be finalized and added to the repository.

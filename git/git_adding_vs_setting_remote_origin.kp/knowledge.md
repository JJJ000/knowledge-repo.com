**Contents**

[TOC]

**Remote Add Origin**

1. Description: `git remote add origin` is used to add a new remote repository. It creates a new connection to a remote repository.

2. Syntax: `git remote add <name> <url>`

**Remote Set-Url Origin**

1. Description: `git remote set-url origin` is used to change the URL of a remote repository. It updates an existing connection to a remote repository.

2. Syntax: `git remote set-url <name> <url>`

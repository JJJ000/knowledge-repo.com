### Using the Command Line
1. Navigate to the local repository directory
2. Run the following command: `git remote -v`
3. The output will be the remote repository origin address

### Using the GitHub Website
1. Log in to the GitHub website
2. Navigate to the repository page
3. Click the "Clone or Download" button
4. The repository URL will be displayed

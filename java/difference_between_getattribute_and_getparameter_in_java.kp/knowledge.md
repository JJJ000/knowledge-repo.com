### getAttribute()
- `getAttribute()` is a method used to retrieve the value of an attribute from an HTML element.
- It is used to access the attributes of an HTML element such as id, class, name, etc.
- It is used to access the values of the attributes defined in the HTML tag.

### getParameter()
- `getParameter()` is a method used to retrieve the value of a parameter from a request.
- It is used to access the parameters of a request such as query string, post data, etc.
- It is used to access the values of the parameters defined in the request.

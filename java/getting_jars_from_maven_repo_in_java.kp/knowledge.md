**Contents**

[TOC]

### Setup

1. Create a Maven project in your preferred IDE.
2. Add the necessary dependencies to your project.

### Download from Maven Repository

1. Navigate to the Maven repository where the source JARs are located.
2. Download the source JARs that you need.

### Add to Project

1. Add the downloaded source JARs to your project.
2. Ensure that the source JARs are included in your project's build path.

### Compile

1. Compile your project using the downloaded source JARs.
2. Verify that the source JARs are correctly included in the project.

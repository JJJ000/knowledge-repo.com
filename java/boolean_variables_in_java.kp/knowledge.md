# Difference
Boolean is a primitive data type in Java while Boolean is a wrapper class.

# Boolean
Boolean is a primitive data type that can only take two values, true or false.

# Boolean
Boolean is a wrapper class that wraps the primitive boolean value and provides methods to parse strings and convert to boolean values. It also provides methods to compare two boolean values.

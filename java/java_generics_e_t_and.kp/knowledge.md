### 'E'

'E' is used to represent an element type in a generic class or method. It is typically used as a parameter for a generic type.

### 'T'

'T' is used to represent a type parameter in a generic class or method. It is typically used to define the type of a generic class or method.

### '?'

'?' is used to represent an unknown type in a generic class or method. It is typically used when the type of a generic class or method is not known.

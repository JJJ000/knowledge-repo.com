### Option 1

1. Select **Preferences** from the **IntelliJ IDEA** menu.
2. Navigate to **Editor** > **General** > **Parameter Info**.
3. Uncheck the **Show parameter name hints** checkbox.

### Option 2

1. Select **File** > **Settings**.
2. Navigate to **Editor** > **General** > **Parameter Info**.
3. Uncheck the **Show parameter name hints** checkbox.

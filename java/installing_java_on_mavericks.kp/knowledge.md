### Downloading Java

1. Go to the [Java downloads page](https://www.java.com/en/download/mac_download.jsp)
2. Click on the **Download** button
3. Double-click the **.dmg** file to launch the installer

### Installing Java

1. Follow the instructions in the installer
2. Click **Install** to begin the installation process
3. Enter your password when prompted

### Configuring Java

1. Open **System Preferences**
2. Click on the **Java** icon
3. Click on the **Security** tab
4. Select the **Enable Java content in the browser** checkbox

### Verifying Installation

1. Open a web browser
2. Go to the [Verify Java Version page](https://www.java.com/en/download/installed.jsp)
3. Follow the instructions to verify the installation

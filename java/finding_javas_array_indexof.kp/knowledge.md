**Contents**

[TOC]

1. **Syntax**
`public int indexOf(Object o)`

2. **Description**
This method returns the index of the first occurrence of the specified element in this array, or -1 if this array does not contain the element.

3. **Parameters**
- o: the element to search for

4. **Returns**
The index of the first occurrence of the specified element in this array, or -1 if this array does not contain the element.

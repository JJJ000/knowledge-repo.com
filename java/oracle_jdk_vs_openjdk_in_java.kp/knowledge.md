### Licensing
Oracle JDK is licensed under the Oracle Binary Code License Agreement which is a proprietary license. OpenJDK is licensed under GPLv2 with the Classpath Exception which is an open source license.

### Support
Oracle JDK is supported by Oracle with long-term commercial support, bug fixes, and security updates. OpenJDK is supported by the open source community with limited commercial support and no bug fixes or security updates.

### Features
Oracle JDK has features such as Java Flight Recorder, Java Mission Control, and commercial support for Java Web Start. OpenJDK does not have these features.

### Availability
Oracle JDK is available for download from Oracle's website. OpenJDK is available for download from a variety of sources, such as AdoptOpenJDK, Amazon Corretto, or Azul Zulu.

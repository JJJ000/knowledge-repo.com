### Path to JDK
The Java Development Kit (JDK) is typically installed in the `/Library/Java/JavaVirtualMachines` directory.

### Path to JRE
The Java Runtime Environment (JRE) is typically installed in the `/Library/Internet Plug-Ins/JavaAppletPlugin.plugin/Contents/Home` directory.

### Path to Java Home
The Java home directory is typically located in the `/Library/Java/Home` directory.

### Path to Java Binaries
The Java binaries are located in the `/usr/bin` directory.

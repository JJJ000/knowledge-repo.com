### Open Eclipse

Open Eclipse and go to the Java perspective.

### Disable Coverage Highlighting

In the menu bar, select Window > Preferences. In the Preferences window, go to Java > Editor > Syntax Coloring. Under the Category list, select Coverage Highlighting. Uncheck the Enable checkbox.

### Save the Changes

Click OK to save the changes.

### Restart Eclipse

Restart Eclipse for the changes to take effect.

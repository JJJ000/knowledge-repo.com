**1. Shortcut**

Ctrl + Alt + ↑/↓ 

**2. Description**

This shortcut will copy the entire line of code, including any leading whitespace or comments, and insert it directly above or below the line of code where the cursor is located.

**3. Platforms**

This shortcut is available for Windows, Mac, and Linux operating systems.

**4. Usage**

This shortcut is useful for quickly copying and pasting large blocks of code or for quickly duplicating a line of code without having to manually highlight and copy the code.

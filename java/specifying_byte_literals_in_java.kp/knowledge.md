# Syntax
Byte literals must be specified with an 'L' suffix, and can be specified in decimal, octal, or hexadecimal formats.

# Decimal
The syntax for a decimal byte literal is: 
`byte b = (byte)L;`

# Octal
The syntax for an octal byte literal is: 
`byte b = (byte)0L;`

# Hexadecimal
The syntax for a hexadecimal byte literal is: 
`byte b = (byte)0xL;`

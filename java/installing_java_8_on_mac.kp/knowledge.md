### Step 1: Download Java 8
1. Go to the [Java SE Downloads page](https://www.oracle.com/java/technologies/javase-downloads.html).
2. Click the **Download** button for the **JDK** (not the JRE).
3. Select the version of Java 8 you want to install.

### Step 2: Install Java 8
1. Open the **.dmg** file you downloaded.
2. Double-click the **.pkg** file in the window that appears.
3. Follow the instructions in the installer.

### Step 3: Verify Installation
1. Open **Terminal**.
2. Type `java -version` and press **Enter**.
3. Verify that the version reported is the one you installed.

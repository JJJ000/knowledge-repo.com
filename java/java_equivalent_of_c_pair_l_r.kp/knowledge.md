### Option 1:
Java does not have an exact equivalent of the C++ Pair<L,R> class.

### Option 2:
A possible equivalent of the C++ Pair<L,R> class in Java is the `AbstractMap.SimpleEntry<K,V>` class.

### Option 3:
Another possible equivalent of the C++ Pair<L,R> class in Java is the `AbstractMap.SimpleImmutableEntry<K,V>` class.

### Option 4:
A third possible equivalent of the C++ Pair<L,R> class in Java is the `org.apache.commons.lang3.tuple.MutablePair<L,R>` class.

### Integer Data Type
In Java, the `int` data type is a 32-bit signed two's complement integer. It has a minimum value of -2<sup>31</sup> and a maximum value of 2<sup>31</sup>-1.

### Maximum Value
Therefore, the maximum value of an integer in Java is 2<sup>31</sup>-1, which is 2,147,483,647.

### Minimum Value
The minimum value of an integer in Java is -2<sup>31</sup>, which is -2,147,483,648.

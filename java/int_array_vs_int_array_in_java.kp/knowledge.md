## Syntax
int[] array: This syntax is used to declare an array of int type.

int array[]: This syntax is used to declare an array of int type by using the traditional method.

## Use
int[] array: This syntax is used to declare an array of int type in the modern Java programming language.

int array[]: This syntax is used to declare an array of int type in the traditional Java programming language.

## Difference
int[] array: This syntax is used to declare an array of int type in a more concise way.

int array[]: This syntax is used to declare an array of int type in a longer way.

### Syntax
`int[][] arrayName = new int[rows][columns];`

### Example
`int[][] myArray = new int[3][4];`

### Explanation
This syntax creates a two-dimensional array called `myArray` that has 3 rows and 4 columns. Each element in the array is an `int` type.

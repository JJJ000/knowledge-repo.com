### ApplicationContext.xml
ApplicationContext.xml is an XML file that contains beans configuration. It is a Spring configuration file that is used to define beans and other related configurations. It is used to create and configure beans. It is used to define the configuration metadata for an application.

### Spring-Servlet.xml
Spring-Servlet.xml is an XML file that contains web-related beans configuration. It is a Spring configuration file that is used to define beans and other related configurations related to web applications. It is used to define the configuration metadata for a web application. It is used to define view resolvers, interceptors, and other web-related configuration.

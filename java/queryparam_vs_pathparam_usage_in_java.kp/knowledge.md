### @QueryParam

@QueryParam is used to bind query parameters in a request URL. It is used to retrieve data from the server. It is generally used in GET requests.

### @PathParam

@PathParam is used to bind path parameters in a request URL. It is used to identify a specific resource in the server. It is generally used in PUT, POST and DELETE requests.

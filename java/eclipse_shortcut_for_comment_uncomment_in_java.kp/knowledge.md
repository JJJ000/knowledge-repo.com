# Shortcut
The shortcut for comment/uncomment in Java in Eclipse is `Ctrl + /`.

# Example
For example, to comment out a line of code, highlight the line and press `Ctrl + /`. To uncomment a line of code, highlight the line and press `Ctrl + /` again.

# Additional Information
In addition, this shortcut works for both single line and multiple line comments. It is also possible to comment/uncomment a block of code by highlighting the block and pressing `Ctrl + /`.

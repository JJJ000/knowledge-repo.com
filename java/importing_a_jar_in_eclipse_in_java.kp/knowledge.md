### Step 1: Add the Jar File to the Project
1. Right-click on the project and select `Build Path` -> `Configure Build Path`.
2. Select the `Libraries` tab and click `Add External JARs`.
3. Select the jar file to be imported and click `Open`.

### Step 2: Refresh the Project
1. Right-click on the project and select `Refresh`.

### Step 3: Verify the Jar File is Imported
1. Right-click on the project and select `Build Path` -> `Configure Build Path`.
2. Select the `Libraries` tab and verify that the jar file is listed.

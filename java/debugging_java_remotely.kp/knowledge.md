**Contents**

[TOC]

## Setting Up
1. Ensure the application is running with the necessary debug parameters.
2. Configure the remote debugger in the IDE of your choice.

## Connecting
1. Connect to the remote debugger with the IDE.
2. Set breakpoints in the application code.

## Debugging
1. Step through the code execution to identify any issues.
2. Make changes in the code as necessary.

## Finishing
1. Disconnect from the remote debugger.
2. Restart the application with the new changes.

**Using \n**

1. Syntax: 
    ```alert("Hello\nWorld!");```
2. Example:
    ```alert("Welcome to\nJavaScript Alerts!");```

**Using HTML**

1. Syntax: 
    ```alert("Hello<br>World!");```
2. Example:
    ```alert("Welcome to<br>JavaScript Alerts!");```

## Syntax

The newline character in JavaScript is `\n`.

## Usage

The newline character can be used to create line breaks within strings. For example:

```
let str = "This is a string\nwith a line break";
```

## Output

The output of the above code would be:

```
This is a string
with a line break
```

## Other Options

The newline character can also be used with the `console.log()` command to create a line break in the console output. For example:

```
console.log("This is a string\nwith a line break");
```

The output of the above code would be:

```
This is a string
with a line break
```

### Window
Window is the top-level object in the JavaScript hierarchy, and it is the global object. It represents a browser window, and it contains all the other objects, such as document, history, location, and screen.

### Document
Document is an object that represents the HTML document that is loaded into the browser. It contains information about the document, such as the title, headings, and body.

### Screen
Screen is an object that represents the physical display of the computer. It contains information about the display, such as the width, height, and color depth.

### Difference
The main difference between window, document, and screen in JavaScript is that window is the global object, document is an object that represents the HTML document, and screen is an object that represents the physical display of the computer.

**Option 1: {}**

**Description:**
The empty object literal notation is a way of creating an empty object in JavaScript. It is denoted by two curly braces containing nothing between them.

**Syntax:**
`{}`

**Example:**
```javascript
let emptyObject = {};
```

**Option 2: new Object()**

**Description:**
The new Object() constructor is a way of creating an empty object in JavaScript. It is denoted by the keyword new followed by the Object() constructor.

**Syntax:**
`new Object()`

**Example:**
```javascript
let emptyObject = new Object();
```

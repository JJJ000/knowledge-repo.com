#Prototype
Prototype is an object that is associated with all JavaScript objects. It is used to add properties and methods to objects. It is also used to create inheritance. 

#This
This is a keyword that is used to refer to the current object in a function or method. It is used to access and manipulate the properties and methods of the current object.

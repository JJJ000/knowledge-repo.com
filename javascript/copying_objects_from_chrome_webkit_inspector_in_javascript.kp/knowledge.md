## Copy Object from Webkit Inspector
1. Right-click on the object in the webkit inspector.
2. Select the "Copy Object" option from the drop-down menu.
3. The object will be copied to the clipboard in the form of a JavaScript object literal.
4. Paste the object into a text editor or IDE to view the code.

### Letters

JavaScript variable names can contain any combination of letters from the English alphabet, both capitalized and lowercase.

### Numbers

JavaScript variable names can contain numbers, as long as the first character is not a number.

### Special Characters

JavaScript variable names can contain special characters such as the underscore (_) and dollar sign ($).

### Reserved Words

JavaScript variable names cannot contain any of the JavaScript reserved words, such as "var", "let", or "function".

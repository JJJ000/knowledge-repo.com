# Section 1: Open the Developer Tools
1. Open the Chrome browser.
2. Right-click anywhere on the page and select **Inspect** from the menu.

# Section 2: Access the Settings
1. Click the **Settings** icon in the lower right corner of the Developer Tools window.

# Section 3: Disable JavaScript
1. In the **Preferences** tab, locate the **Disable JavaScript** option.
2. Select the **Disable JavaScript** checkbox.

# Section 4: Save Changes
1. Click the **Done** button to save your changes.

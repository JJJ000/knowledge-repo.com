**Removing an Attribute from a JSON Object**

1. Using the `delete` Operator:
    - The `delete` operator can be used to remove an attribute from a JSON object.
    - Syntax: `delete objectName.attributeName`

2. Using the `splice` Method:
    - The `splice` method can be used to remove an attribute from a JSON object.
    - Syntax: `objectName.splice(index, 1)`

3. Using the `pop` Method:
    - The `pop` method can be used to remove an attribute from a JSON object.
    - Syntax: `objectName.pop(attributeName)`

4. Using the `shift` Method:
    - The `shift` method can be used to remove an attribute from a JSON object.
    - Syntax: `objectName.shift(attributeName)`

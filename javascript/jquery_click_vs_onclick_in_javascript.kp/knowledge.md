#### jQuery.click()
jQuery.click() is a jQuery method that is used to bind an event handler to the click JavaScript event, or trigger that event on an element.

#### onClick in Javascript
onClick is an event handler that is supported by all major browsers and is used to execute a script when an element is clicked.

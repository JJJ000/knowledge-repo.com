**Contents**

[TOC]

__proto__
- Definition: 
__proto__ is an accessor property that returns the prototype of the object it is called on.

- Syntax:
object.__proto__

- Usage:
__proto__ is used to access the prototype of an object.

Prototype
- Definition: 
Prototype is an object that is used as a template for creating new objects.

- Syntax:
object.prototype

- Usage:
Prototype is used to create new objects and set the prototype of the newly created object.

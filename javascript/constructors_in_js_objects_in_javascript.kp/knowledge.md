1. Constructor Function
A constructor function is a special type of function used to create an object. It is used to create a new instance of an object and can be used to set properties and methods for the object. The constructor function is usually named with a capital first letter.

2. Constructor Method
A constructor method is a special type of method used to create an object. It is used to create a new instance of an object and can be used to set properties and methods for the object. The constructor method is usually named with a capital first letter.

3. Object Literal
Object literals are a way to create objects without using a constructor function or method. It is a simple way to create an object by defining the properties and methods of the object in a single line of code.

4. Object Constructor
The Object constructor is a built-in constructor function used to create an object. It is used to create a new instance of an object and can be used to set properties and methods for the object. The Object constructor is usually named with a capital first letter.

# Section 1: Access the Console
1. Open Google Chrome and navigate to the website you want to debug.
2. Right-click anywhere on the page and select "Inspect" from the context menu.
3. Click the "Console" tab at the top of the window that appears.

# Section 2: Enable Persistent Logging
1. Click the "Settings" icon in the upper right corner of the Console window.
2. Check the "Preserve Log upon Navigation" box.

# Section 3: Use the Console
1. Type your JavaScript code into the console.
2. Click the "Enter" key or press the "Run" button to execute the code.

# Section 4: Review the Output
1. Check the output in the Console window.
2. If the output is what you expected, the code was successful. If not, debug the code and make adjustments until it works as desired.

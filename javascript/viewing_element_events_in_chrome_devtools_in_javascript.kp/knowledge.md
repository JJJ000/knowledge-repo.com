--------------------------------

2. Select Elements Tab
--------------------------------

3. Select Element
--------------------------------

4. Navigate to Event Listeners Tab
--------------------------------
The Event Listeners tab will show all events fired on the selected element. In the Event Listeners tab, you can also view the source code for each event listener.

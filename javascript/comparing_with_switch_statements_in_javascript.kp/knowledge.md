1. Syntax
```
switch (expression) {
  case x:
    // code block
    break;
  case y:
    // code block
    break;
  default:
    // code block
}
```

2. Greater Than
```
switch (expression) {
  case expression > x:
    // code block
    break;
  default:
    // code block
}
```

3. Less Than
```
switch (expression) {
  case expression < x:
    // code block
    break;
  default:
    // code block
}
```

4. Greater Than or Equal To
```
switch (expression) {
  case expression >= x:
    // code block
    break;
  default:
    // code block
}
```

# Section 1: What is Typescript?
Typescript is a typed superset of JavaScript that compiles to plain JavaScript. It adds optional static typing to the language and provides support for the latest JavaScript features, including those from ES6, ES7, and ES8.

# Section 2: What is a String?
In programming, a string is a sequence of characters, such as a word, a phrase, or a text string. Strings are commonly used in programming languages to represent text-based data.

# Section 3: Typescript and Javascript
In Typescript, a string is a type that represents textual data. It is not assignable to type in Javascript, as Javascript does not have a type system. 

# Section 4: Conclusion
In conclusion, Typescript type 'string' is not assignable to type in Javascript, as Javascript does not have a type system. However, Typescript provides support for the latest JavaScript features, including those from ES6, ES7, and ES8.

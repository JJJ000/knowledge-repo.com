1. Check for Touch Events
The most reliable way to detect a touch screen device is to check for the presence of touch events. This can be done by checking the `ontouchstart` event in the `window` object.

```js
if ('ontouchstart' in window) {
    // device is a touch screen
} else {
    // device is not a touch screen
}
```

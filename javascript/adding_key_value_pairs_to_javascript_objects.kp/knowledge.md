**Using Dot Notation**

1. Create an object: 

```javascript
let myObj = {};
```

2. Add a key/value pair to the object using dot notation: 

```javascript
myObj.key = "value";
```

**Using Square Bracket Notation**

1. Create an object: 

```javascript
let myObj = {};
```

2. Add a key/value pair to the object using square bracket notation: 

```javascript
myObj["key"] = "value";
```

### Array.length

Array.length is a property of the array object in Javascript that returns the number of elements in the array. It is a read-only property and cannot be used to modify the length of the array.

### Array.size()

Array.size() is a method of the array object in Javascript and is not a standard Javascript method. It is a custom method created by developers to return the number of elements in the array. It is not supported in all browsers and can be used to modify the length of the array.

# Angular-Route
Angular-route is an AngularJS module that is used to create single page applications. It provides the ability to create multiple views within a single page and to switch between them based on user interactions. It supports routing and deep linking, which allows users to bookmark and share specific views.

# Angular-UI-Router
Angular-ui-router is an AngularJS module that provides advanced routing and state management. It allows for nested views and multiple named views, which makes it easy to create complex user interfaces. It also supports multiple named views, which allows for more flexibility when creating complex user interfaces. It also supports lazy loading, which allows for a more efficient loading of resources. Finally, it supports URL routing, which allows for better SEO optimization.

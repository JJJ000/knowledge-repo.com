# Windows

1. Open Start Menu
2. Search for "Python"
3. Select "Python 3.X" from the list
4. Right-click on the Python icon and select "Properties"
5. The "Target" field in the "General" tab will give you the full path of the Python interpreter

# Mac

1. Open Finder
2. Go to Applications
3. Double-click on the Python 3.X folder
4. Right-click on the Python icon and select "Get Info"
5. The "Where" field in the "General" tab will give you the full path of the Python interpreter

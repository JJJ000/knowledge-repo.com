# Syntax

The syntax for an inline if statement in Python is as follows:

```python
print('True' if condition else 'False')
```

# Example

Here is an example of an inline if statement for printing in Python:

```python
x = 10
print('x is greater than 5' if x > 5 else 'x is not greater than 5')
```

# Output

This code will print the following:

`x is greater than 5`

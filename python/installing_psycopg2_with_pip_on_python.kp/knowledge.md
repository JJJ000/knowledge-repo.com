# Installation

1. Install pip if not already installed.
2. Open a command prompt and navigate to the folder containing the pip installer.
3. Run the command `pip install psycopg2`

# Verification

1. Open a command prompt and type `python`
2. Type `import psycopg2`
3. If no errors occur, the installation was successful.

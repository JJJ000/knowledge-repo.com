# Section 1: Using the ord() Function
The ord() function is used to get the ASCII value of a character in Python.

Syntax:

ord(character)

# Section 2: Example
Below is an example of how to use the ord() function to get the ASCII value of a character in Python.

```
# Get the ASCII value of character 'A'
ascii_value = ord('A')

# Print the ASCII value
print(ascii_value)

# Output
65
```

# Section 3: Using the chr() Function
The chr() function can also be used to get the ASCII value of a character in Python.

Syntax:

chr(ascii_value)

# Section 4: Example
Below is an example of how to use the chr() function to get the ASCII value of a character in Python.

```
# Get the ASCII value of character 'A'
ascii_value = chr(65)

# Print the ASCII value
print(ascii_value)

# Output
A
```

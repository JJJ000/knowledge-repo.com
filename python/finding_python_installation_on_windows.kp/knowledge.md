#1 Using the Command Prompt
1. Open the Command Prompt. 
2. Type `where python` and press Enter. 
3. The path to the Python executable will be displayed.

#2 Using the Windows Search
1. Open the Windows Search.
2. Type `python` and press Enter. 
3. Select the Python executable from the list of results. 
4. Right-click the Python executable and select Properties. 
5. The path to the Python executable will be displayed in the Properties window.

#### print()
The most basic way to achieve a similar effect as `var_dump()` in Python is to use the `print()` function. This will print the contents of any object to the console.

#### type()
The `type()` function can be used to print the type of an object.

#### pprint()
The `pprint()` function can be used to print a more readable version of a complex object.

#### dir()
The `dir()` function can be used to print all the attributes and methods of an object.

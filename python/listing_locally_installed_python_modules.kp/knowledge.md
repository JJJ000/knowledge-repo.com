## Using pip

1. Open a terminal window
2. Type `pip freeze`
3. This will list all the installed packages and their versions

## Using the Python Interpreter

1. Open a terminal window
2. Type `python` to open the Python interpreter
3. Type `help("modules")`
4. This will list all the installed modules

## Using the py command

1. Open a terminal window
2. Type `py -m pip list`
3. This will list all the installed packages and their versions

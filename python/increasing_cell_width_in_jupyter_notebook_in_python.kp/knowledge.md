1. **Using the Menu Bar**

- Go to the View menu
- Select “Toggle Line Wrapping”
- This will expand the cell width to the full width of the browser window

2. **Using Keyboard Shortcuts**

- Press “Ctrl + Shift + P”
- Type “line wrapping”
- Select “Toggle Line Wrapping”

3. **Using the Command Palette**

- Press “Ctrl + Shift + P”
- Type “command palette”
- Select “Toggle Line Wrapping”

4. **Using the Toolbar**

- Click the “Toggle Line Wrapping” button on the toolbar
- This will expand the cell width to the full width of the browser window

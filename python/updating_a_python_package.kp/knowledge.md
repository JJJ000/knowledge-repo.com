# Updating Python Packages

## Using pip
1. Open a terminal window and type `pip install --upgrade PackageName`.
2. Press enter and the package will be updated to the latest version.

## Using conda
1. Open a terminal window and type `conda update PackageName`.
2. Press enter and the package will be updated to the latest version.

## Using Anaconda Navigator
1. Open Anaconda Navigator and select the Environments tab.
2. Select the environment containing the package you want to update.
3. Select the package you want to update and click the arrow next to it.
4. Select “Update selected packages” and click Apply.

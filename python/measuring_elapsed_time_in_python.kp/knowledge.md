## Using the Time Module

The `time` module can be used to measure elapsed time in Python. This module provides various functions, such as `time()`, `sleep()`, and `clock()`, which can be used to measure the elapsed time of a program.

## Using the Datetime Module

The `datetime` module can also be used to measure elapsed time in Python. This module provides various classes, such as `datetime`, `date`, and `timedelta`, which can be used to measure the elapsed time of a program.

## Using the Timeit Module

The `timeit` module is another way to measure elapsed time in Python. This module provides various functions, such as `timeit()`, which can be used to measure the elapsed time of a program.

## Using the Timer Module

The `timer` module is a fourth way to measure elapsed time in Python. This module provides various functions, such as `Timer()`, which can be used to measure the elapsed time of a program.

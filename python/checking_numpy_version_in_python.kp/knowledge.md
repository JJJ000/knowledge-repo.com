# Using Command Line

1. Open your command line.
2. Type `pip list` and press enter.
3. Look for the version of NumPy in the list of installed packages.

# Using Python

1. Open your Python console.
2. Type `import numpy` and press enter.
3. Type `numpy.__version__` and press enter.
4. The version of NumPy will be displayed.

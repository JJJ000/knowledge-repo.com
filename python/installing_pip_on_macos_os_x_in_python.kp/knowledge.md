### Downloading Pip
1. Download the get-pip.py file from https://bootstrap.pypa.io/get-pip.py
2. Open a Terminal window and navigate to the folder containing the get-pip.py file.

### Installing Pip
1. Run the following command in the Terminal window:
    ```python get-pip.py```
2. This will install pip and its dependencies.

### Verifying Installation
1. To verify that pip is installed properly, run the following command in the Terminal window:
    ```pip -V```
2. This should display the version of pip installed on your system.

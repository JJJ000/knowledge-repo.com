# Using pip
1. Open the command line interface in your system.
2. Enter the command `pip list` to view all the installed modules and their versions.

# Using Python
1. Open the Python interpreter.
2. Enter the command `help("modules")` to view all the installed modules.
3. Enter the command `help("<module_name>")` to view the version of the particular module.

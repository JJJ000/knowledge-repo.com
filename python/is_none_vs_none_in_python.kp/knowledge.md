##### "is None"

`is None` is a comparison operator that compares the identity of two objects. It returns `True` if both objects are the same object, and `False` if they are not.

##### "== None"

`== None` is a comparison operator that compares the values of two objects. It returns `True` if the objects have the same value, and `False` if they do not.

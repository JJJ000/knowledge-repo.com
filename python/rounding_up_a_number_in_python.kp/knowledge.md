# Round Up Function
The round() function in Python returns a number rounded to the specified number of digits. To round up a number, use the math.ceil() function.

# Syntax
The syntax for the ceil() function is:

math.ceil(x)

# Parameters
x -- This is a numeric expression.

# Return Value
The ceil() function returns the smallest integer value greater than or equal to x.

#1 Using type()

The type() function returns the type of the passed variable. This function can be used to get the class name of an instance in Python.

#2 Using __class__

The __class__ attribute of an instance of a class returns the class to which it belongs. This can be used to get the class name of an instance in Python.

#3 Using isinstance()

The isinstance() function returns True if the specified object is of the specified type, otherwise False. This can be used to get the class name of an instance in Python.

#4 Using getattr()

The getattr() function returns the value of the specified attribute of the specified object. This can be used to get the class name of an instance in Python.

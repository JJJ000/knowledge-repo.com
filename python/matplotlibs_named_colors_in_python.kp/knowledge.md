## Basic Colors

- Blue: `'b'`
- Green: `'g'`
- Red: `'r'`
- Cyan: `'c'`
- Magenta: `'m'`
- Yellow: `'y'`
- Black: `'k'`
- White: `'w'`

## Hex Colors

- Aqua: `'#00ffff'`
- Blue: `'#0000ff'`
- Fuchsia: `'#ff00ff'`
- Gray: `'#808080'`
- Lime: `'#00ff00'`
- Maroon: `'#800000'`
- Navy: `'#000080'`
- Olive: `'#808000'`
- Purple: `'#800080'`
- Silver: `'#c0c0c0'`
- Teal: `'#008080'`

## RGB Colors

- Blue: `(0, 0, 255)`
- Green: `(0, 255, 0)`
- Red: `(255, 0, 0)`
- Cyan: `(0, 255, 255)`
- Magenta: `(255, 0, 255)`
- Yellow: `(255, 255, 0)`
- Black: `(0, 0, 0)`
- White: `(255, 255, 255)`

## CMYK Colors

- Blue: `(100, 100, 0, 0)`
- Green: `(100, 0, 100, 0)`
- Red: `(0, 100, 100, 0)`
- Cyan: `(0, 0, 0, 0)`
- Magenta: `(0, 100, 0, 0)`
- Yellow: `(0, 0, 100, 0)`
- Black: `(0, 0, 0, 100)`
- White: `(0, 0, 0, 0)`

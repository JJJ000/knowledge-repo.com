# *args
* *args is used to pass a variable number of arguments to a function.
* It is used to pass a non-keyworded, variable-length argument list.
* *args is used as a parameter when we do not know how many arguments need to be passed to a function.

# **kwargs
* **kwargs is used to pass a keyworded, variable-length argument list. 
* It allows us to pass the variable number of keyword arguments to a function.
* The keyword arguments return a dictionary in which the keys are the argument names and the values are the argument values.
* **kwargs can be used to pass arbitrary keyword arguments to a function.

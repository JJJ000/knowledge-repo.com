Yes, there is.

#### Syntax
The syntax for the "not equal" operator in Python is `!=`.

#### Usage
The "not equal" operator is used to compare two values and returns `True` if the values are not equal, and `False` if they are equal.

#### Example
For example, the following code will return `True`:
```
a = 5
b = 6

a != b
```

# Setting Maximum Line Length

## Method 1: Using PyCharm Preferences
1. Open the PyCharm Preferences window (**File > Settings** on Windows and Linux, **PyCharm > Preferences** on macOS).
2. In the left panel, select **Editor > Code Style > Python**.
3. In the right panel, select the **Wrapping and Braces** tab.
4. In the **Right margin (columns)** field, enter the desired maximum line length.

## Method 2: Using Editor Settings
1. Open the PyCharm Editor Settings window (**File > Settings > Editor > General** on Windows and Linux, **PyCharm > Preferences > Editor > General** on macOS).
2. In the right panel, select the **Soft Wraps** tab.
3. In the **Right margin** field, enter the desired maximum line length.
